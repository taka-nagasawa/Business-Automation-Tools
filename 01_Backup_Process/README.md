# 01_Backup_Process（前日ファイルの退避処理）

## 1. 概要
本バッチファイルは、業務を開始する前に、前日に作成されたファイルを自動的にバックアップ用フォルダ（old）へ移動させるツールです。

## 2. 解決したい課題
* 毎日手作業でファイルを移動させる手間の削減。
* 前日のファイルを誤って上書きしたり、消してしまったりするミスを防ぐ。

## 3. 動作内容
1. バッチを実行した日から、「昨日の日付」を特定します。
2. バッチと同じフォルダ内に `old` フォルダがあるか確認し、なければ自動で作ります。
3. 昨日の日付がついたファイルを探して、`old` フォルダの中へ移動させます。
   
## 4. 動作確認の手順（テスト方法）
1. 任意のフォルダ内に、本バッチファイル（backup_yesterday.bat）を保存します。
2. 同じフォルダ内に、昨日の日付の名前をつけたExcelファイルを作成します。
   （例：今日が2026年2月19日の場合、`20260218.xlsx` というファイルを作成）
3. バッチファイルをダブルクリックして実行します。
4. 自動的に `old` フォルダが作成され、その中にExcelファイルが移動していれば成功です。
