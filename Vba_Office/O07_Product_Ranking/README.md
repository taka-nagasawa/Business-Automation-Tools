# O07 お弁当注文ランキング

## 概要
バラバラに入力された「注文リスト」から、メニューごとの個数を集計し、注文が多い順に並べ替えます。

## 使い方
### **【重要：実行前の準備】**
ダウンロード後、ファイルを右クリック→「プロパティ」→「許可する」にチェックを入れてください。

### **【シート別の説明】**
- **注文リスト**: B列に注文したメニュー名が入っているシートです。
- **集計結果**: 実行すると自動で作成され、人気メニュー順に並びます。

### **【注意】**
- 実行前に、すでに「集計結果」というシートがないか確認してください（あるとエラーになります）。
- マクロ実行後の「元に戻す」はできません。

### **実行手順**
1. 「注文リスト」シートでマクロを実行。
2. どの弁当をいくつ注文すべきか、注文数が多い順に表示されます。

## 特徴
- メニュー名が増えても、自動でダブりを消してリストを作ります。
- COUNTIF関数を使って自動集計するので、手作業より正確です。
- 最後に注文数でソートをかけるので、一番人気のメニューがすぐ分かります。
