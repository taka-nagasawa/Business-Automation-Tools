# O10 メール作成・送信（Outlook連携）

### 概要
Excelシートに「宛先」「件名」「本文」「送信可否」を入力し、ボタン一つでOutlookメールをまとめて作成・送信するツールです。宛先ごとに内容が異なる大量のメール送信を自動化し、コピペ作業を排除します。

### マクロ使用前の注意
- **ダウンロード時の重要設定**
    - ファイルを開いた際に表示される「コンテンツの有効化」を必ずクリックしてください。
    - ファイルは「.xlsm（マクロ有効ブック）」形式で保存・管理してください。
- **直接コード修正による送信モード切替**
    - 本ツールは、用途に合わせて**VBAコードを直接修正（書き換え）**して使用します。
    - **【確認モード】**（初期設定）：
        `.Display` が有効な状態です。送信前に1件ずつメール画面が表示されます。
    - **【直接送信モード】**（一括送信）：
        VBAエディタを開き、`.Display` の左側に `'`（シングルクォート）を書き加えて無効化し、逆に `.Send` の左側にある `'` を消して有効化することで、画面表示なしで一括送信されるようになります。
- **処理時間の理由**
    - 1件ずつExcelからOutlookへデータを渡してメールを組み立てる通信が発生するため、件数が多いと完了まで数分かかる場合があります。

### 操作の手順
- **テストデータの確認**
    - 「data」シートには、あらかじめリスト形式で**テストデータが入力されています**。まずはこのデータを活用して動作を確認してください。
- **メール情報の入力と選択**
    - 宛先・件名・本文を整え、送信したい行のD列（送信可否）を「OK」にします。
- **一括作成の実行**
    - 「メール作成」ボタンを押すと、OKにした行のメールが自動で作成（または送信）されます。

### シート別の説明
- **data**
    - メインの入力シートです。ここに入力された内容がそのままメールになります。
- **リスト**
    - 宛先や定型文など、入力の手間を省くためのマスターデータを管理するシートです。
