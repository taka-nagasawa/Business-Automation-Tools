# O05 特定キーワードの行抽出

## 概要
「データ一覧」シートから、指定した条件に合う行を丸ごと「抽出結果」シートへ抜き出します。
標準では「重要」を抽出しますが、コードを書き換えればどんな言葉でも抽出可能です。

## 使い方
### **【重要：実行前の準備】**
ダウンロード後、ファイルを右クリック→「プロパティ」→「許可する」にチェックを入れてください。

### **【シートの準備】**
- **データ一覧**: 元データを入れておくシートです。
※1行目の見出しは自動でコピーされます。

### **【カスタマイズ】**
コード内の `"*重要*"` の部分を、抽出したい任意の言葉（例：`"*至急*"`、`"*完了*"` など）に変更して使用してください。

### **【注意】**
実行後の「元に戻す」はできません。必ずバックアップを取ってください。

### **実行手順**
1. 「データ一覧」でボタンをクリック。
2. 「抽出結果」シートに該当データのみが転記されます。

## 特徴
**全列自動整形**: 抽出後、すべての列幅をデータに合わせて自動で整えます。
