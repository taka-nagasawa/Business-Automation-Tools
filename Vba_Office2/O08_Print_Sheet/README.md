# O08 健診結果印刷

### 概要
表示中のシートをプリンターへ出力します。特定の番号を指定して1枚ずつ発行する実務向けの構成です。
「健診データ」シートにデータを追加することで、印刷対象を増やすことが出来ます。

### マクロ使用前の準備
- **保存形式の確認**
    - ファイルを保存するときは、必ず「マクロ有効ブック（.xlsm）」の形式にしてください。
- **マクロの有効化**
    - ファイルを開いた際、警告が表示された場合は「コンテンツの有効化」を行ってください。

### シート別の説明
- **印刷シート**
    - J1セルに検索キー（患者番号）を入力してEnterを押すことで健診データを表示させます。
    - 印刷ボタンをこのシートに配置します。
- **健診データ**
    - A列からT列までを使用したマスターデータシートです。

### 操作の手順
- **患者番号入力**
    - J1セルに直接患者番号を打ち込みEnterキーを押すことで、健診結果が表示されます。
- **印刷実行ボタン**
    - クリック1回で印刷します。
- **データを追加できます**
    - 「健診データ」シートに行を追加するだけで、自動的に印刷対象として認識されます。
