# O09 フォルダ内一括PDF変換（仕分け機能付）

### 概要
フォルダ内のExcelファイルを一括でPDFへ変換し、専用フォルダへ自動で整理・格納する時短ツールです。数に関わらず、フォルダ内のすべての「.xlsx」ファイルを一括処理します。

### マクロ使用前の注意
- **【重要】実行場所について（エラー回避）**
    - 必ず、**「ドキュメント」や「デスクトップ」など、自分のPC上のフォルダ（ローカル環境）**で実行してください。
    - **理由**: OneDriveなどのクラウド上の場所やネットワーク掲示板のような場所で実行すると、通信の影響でエラーが発生しやすいためです。
- **マクロの有効化**
    - 画面上部に「セキュリティの警告」が出た場合は、「コンテンツの有効化」を押してください。
- **処理時間について**
    - ファイル数が多い場合、完了まで数分かかることがあります。
    - **理由**: 1ファイルごとに「Excelを裏で開く」「PDFとして書き出す」「閉じる」という工程を順番に繰り返しているため、PCに負荷がかかります。

### 操作の手順
- **作業用フォルダの準備**
    - 本マクロファイルと、PDF化したいExcelファイルだけを入れた専用フォルダを自分のPC内に作成し、その中で実行してください。
- **テスト用ファイルの作成**
    - 手元にファイルがない場合、ボタン一つでデータ入りExcelを10個生成できます。動作確認にご活用ください。
- **PDF一括出力操作**
    - ボタンをクリックすると自動でPDF変換が始まります。完了メッセージが出るまで画面が静止することがありますが、そのままお待ちください。
- **自動フォルダ仕分け**
    - 同階層に「PDF」フォルダが自動作成されます。
