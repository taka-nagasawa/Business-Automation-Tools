# O01 売上月次集計

売上データを月ごとに集計するマクロを作成しました。

日付と売上金額が入力された一覧から、
年月ごとの売上合計を自動で出力します。

毎回フィルターやピボットを作らなくても、
ボタン1つで月次集計できるようにしています。

---

■ シート準備

以下のシートを作成してください。

・売上データ
・月次集計

---

■ シート構成

【売上データ】
A列：日付  
B列：担当者  
C列：商品名  
D列：売上金額  
※2行目以降にデータを入力

【月次集計】
A列：年月  
B列：売上合計  

---

■ 使い方

1. 売上データシートにデータを入力
2. VBA画面で標準モジュールを作成し、
   コードを貼り付ける
※必要に応じてボタンを配置して下さい
3. マクロ「売上月次集計」を実行
4. 月次集計シートに結果が出力されます
