# O05 重要データの別シート抽出

## 概要
「データ一覧」シートから「重要」というキーワードを含む行を抽出し、新シート「抽出結果」へ転記します。

## 使い方
### **【重要：実行前の準備】**
**本ファイルはマクロが含まれているため、ダウンロード後に以下の操作を行ってください。**
1. ファイルを右クリックして「プロパティ」を開く。
2. 「全般」タブの下部にある「許可する」にチェックを入れて「OK」を押す。

### **【シートの説明】**
本ツールは以下のシート構成で動作します。
- **データ一覧**: 元データを入力しておくシートです。
- **抽出結果**: マクロ実行時に自動で作成されます。同名のシートがある場合は削除してから実行してください。

### **【注意】**
**本マクロは実行後に「元に戻す(Ctrl+Z)」ができません。実行前に必ずファイルのコピー（バックアップ）を取ってから使用してください。**

### **実行手順**
1. 「データ一覧」シートを開きます。
2. シート内に配置されている**「実行ボタン」**をクリックします。
3. 自動的に「抽出結果」シートが作成され、A列の幅が整えられた状態で転記されます。

## 特徴
- **列幅の自動調整**: 抽出完了後、A列の幅を内容に合わせて自動調整するため、すぐに内容を確認できます。
- **柔軟なキーワード判定**: 「重要」という文字がどこかに含まれていれば抽出対象とします。
