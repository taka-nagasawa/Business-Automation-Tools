# O05 重要データの別シート抽出

## 概要
「データ一覧」シートのA列を確認し、「重要」というキーワードを含む行を**行まるごと（横の列もすべて）**、新シート「抽出結果」へ転記します。

## 使い方
### **【重要：実行前の準備】**
**本ファイルはマクロが含まれているため、ダウンロード後に以下の操作を行ってください。**
1. ファイルを右クリックして「プロパティ」を開く。
2. 「全般」タブの下部にある「許可する」にチェックを入れて「OK」を押す。

### **【シートの準備】**
本ツールは以下のシート構成で動作します。
- **データ一覧**: 元データを入力しておくシートです。A列のキーワードを判別し、その行全体をコピーします。
- **抽出結果**: 実行時に作成されます。該当行のデータがすべて転記されます。

### **【注意】**
**本マクロは実行後に「元に戻す(Ctrl+Z)」ができません。実行前に必ずファイルのコピー（バックアップ）を取ってから使用してください。**

### **実行手順**
1. 「データ一覧」シートを開きます。
2. シート内に配置されている**「実行ボタン」**をクリックします。
3. 一瞬で画面が切り替わり、抽出結果が表示されます。

## 特徴
- A列だけでなく、右側に続く備考や担当者などのデータも漏らさずセットで転記します。
- 画面のチラつきを止め、一瞬で結果を表示します。
- 実行ボタンなどの図形はコピーせず、純粋なデータのみを綺麗に貼り付けます。
