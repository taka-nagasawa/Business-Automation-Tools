# O06 担当者別集計

## 概要
「データ一覧」シートから担当者名を抜き出し、個別の合計売上を算出して「集計結果」シートにまとめます。

## 使い方
### **【重要：実行前の準備】**
ダウンロード後、ファイルを右クリック→「プロパティ」→「許可する」にチェックを入れてください。

### **【シートの準備】**
- **データ一覧**: B列に「担当者名」、C列に「金額」が入っている必要があります。（テストデータあり）
- **集計結果**: 実行時に新規作成されます（見出し等は自動コピーされます）。

### **【カスタマイズ】**
コード内の `SUMIF` の参照列（B:BやC:C）を変えるだけで、商品別や拠点別の集計にも転用できます。

### **【注意】**
実行後の「元に戻す」はできません。必ずバックアップを取ってください。

### **実行手順**
1. 「データ一覧」でボタンをクリック。
2. 担当者ごとに集計された新しいシートが作成されます。
